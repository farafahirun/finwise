# Features Overview

## Authentication

- User registration
- User login
- Session management
- Profile page with personal summary

## Financial Risk Assessment

- Input financial data from the main app
- Ratio-based feature engineering
- Machine learning prediction
- Risk label output: Aman, Waspada, Berbahaya

## Dashboard Analytics

- Financial Health Score
- Debt ratio analysis
- Saving rate analysis
- Risk distribution chart
- Trend analysis for debt and saving
- Prediction history table

## AI Financial Advisor

- Financial summary generation
- Chat-based question answering
- Context-aware recommendations
- Persistent conversation history

## Financial Planning

- Financial goals storage
- Emergency fund estimation
- Goal progress monitoring
- Simple monthly saving plan recommendation

## Reporting

- PDF report generation
- Personal summary based on prediction history
- Downloadable financial report
